## Інсталяція Times New Roman для Ubuntu
> Для компіляції проєкту потрібен шрифт Times New Roman. Якщо шрифт не встановлений для вашої системи, згенерований документ буде порожнім. Щоб встановити шрифти:
```
sudo apt-get update
```
```
sudo apt-get --reinstall install ttf-mscorefonts-installer
```


## Інсталяція TeX в Ubuntu [[відео](https://www.youtube.com/watch?v=ioy7e7fxjHo)]

- Інсталяція Tex Live:
 ```
 sudo apt-get install texlive-full
 ```
- Інсталяція редактора Texmaker: 
```
sudo apt-get install texmaker
```


## Збір проєкту LaTex
> Для компіляції всього проєкту  потрібно виконати `start-build`:

- Крок 1 - Дозволяємо виконання `start-build`
```
chmod u+x start-build
```

- Крок 2 - Збираємо проєкт
```
./start-build
```
> `full-diploma.pdf` &mdash; фінальний документ, який Ви отримаєте після збору проєкту LaTex. Цей документ є об'єднанням усіх окремих частин, а саме: титульний аркуш, завдання на дипломний проєкт, опис альбому, технічне завдання, пояснювальна записка та схеми дипломного проєкту (принципова, функціональна, сруктурна).


## Основні відомості для заповнення

Основні дані для заповнення документа знаходяться у файлі [`setup/data.tex`](setup/data.tex).

Відредагуйте цей файл перед початком роботи. Він містить:

- **Автор** — ПІБ студента, курс, група, код у штампі
- **Тема та спеціальність** — назва дипломного проєкту, освітня програма, спеціальність, ступінь
- **Навчальний заклад** — університет, факультет, кафедра
- **Керівник** — ПІБ, посада, науковий ступінь
- **Рецензент та консультант** — ПІБ, посада, науковий ступінь
- **Календарний план** — етапи виконання дипломного проєкту з термінами тощо
