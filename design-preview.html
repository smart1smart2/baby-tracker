Key changes from current

  • Unified kind selector: all 4 forms now use the same tile chrome (rounded card with circle-icon + label, soft violet wash on active). Replaces inconsistent pills/grids.
  • Section label: small uppercase "TYPE" caption above selectors so the role of each block is explicit.
  • Sleep/Feeding active state: when a timer is running, primary CTA + active banner are at the top; manual entry is collapsed below a divider with "Save" demoted to secondary (disabled while a timer is active).
  • Sleep/Feeding idle state: big primary "▶︎ Start now" button replaces the small label-on-input field for quick-start UX.
  • Pickers: date/time inputs use a chevron affordance, full datetime "29 Apr, 14:35" instead of date-only.
  • Buttons: rounded radius 24, consistent paddingY across all forms.
